Installation Guide

Requirements:

    Internet Access
    Firefox Browser (or any web browser that supports bookmarklets)
    Signed into the necessary account in your browser (e.g., Reading Plus, Clever, or similar platforms)

Step 1: Download the Required File

    Open your web browser and navigate to my GitHub page: https://github.com/SAPH1TE/1017274812.
    On the right-hand side of the page, locate the Releases section.
    Find and download the file named "loader.txt", which is positioned directly below the large paragraph I’ve written.
        To download the file, simply click on the blue name of the file.

Step 2: Open the File in Notepad

    After downloading the file, press the Windows key, type Notepad in the search bar, and open the Notepad application.
    In Notepad, go to the File menu (top-left corner) and select Open.
    Navigate to your Downloads folder and open the "loader.txt" file.
    Do not modify the contents of the file. It should look something like this: "javascript:'use%20strict'%3" and so on.
    Leave Notepad open for the next steps.

Step 3: Create a Bookmark in Your Browser

    In your browser (e.g., Firefox, Chrome, Safari), create a new bookmark.
    In the URL field of the bookmark, paste the contents from the loader.txt file that is currently open in Notepad.
    After pasting, save the bookmark and close Notepad.

Step 4: Use the Bookmarklet

    Open the website or platform where you want to use the bookmarklet (e.g., Clever, Reading Plus, or any other supported platform).
    Once on the desired page, click on the bookmark you created. This will execute the bookmarklet and provide the functionality it offers.

Advanced Usage

Custom JavaScript Integration

The script provides a powerful custom JS injection system. Here's how to use it:

// Example: Create a custom automation
const customAutomation = () => {
    // Your custom logic here
    console.log("Custom automation running");
};

// Add it to the sections object
sections.section3.push({
    label: 'Custom Automation',
    id: 'customAuto',
    customJS: customAutomation.toString()
});
